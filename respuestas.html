¿Qué significa HTML y por qué se dice que es un lenguaje de marcas? Enumera 8 etiquetas HTML
Por sus siglas, HTML significa Lenguaje de Marcado de Hipertexto, y se lo considera un lenguaje de marcas debido a su funcion que consiste en encerrar, mediante una serie de elementos, diferentes partes de lo que contiene para manipularlo a gusto, haciendo que se vea o modifique de distintas maneras.
Las distintas etiquetas que hay son: script, head, body, meta, tittle, h1, p, br.

¿Cuál es la anatomía básica del HTML? Escribir su código y definir
La anatamoia basica del HTML consiste en las etiquetas de cierrre (< >), la etiqueta de cierre (/), y el contenido el cual iria, conformando estos tres el elemento.
Su definicion es:
<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta http-equiv="X-UA-Compatible" content="IE=edge"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Document</title> </head> <body> </body> </html>

¿Qué es y que función cumple el CSS?
CSS es un estilo de hojas en cascadas, o mejor dicho, un lenguaje informatico cuya funcion es especificar la forma de presentar los documentos a los usuarios, significando asi la forma en que se diseñan y compaginan, utilizando estilos de texto basicos, tal como el poder cambiar el color o el tamaño.
